# 1С DevOps Pipeline

CI/CD-конвейер для автоматизации тестирования конфигураций 1С:Предприятие с использованием Docker, Vanessa-Automation и GitLab CI/CD.

Выпускная квалификационная работа Филимонова И.Д., МТУСИ, 2026.

## Структура репозитория

- `config/` — демонстрационная конфигурация 1С в формате XML
- `tests/` — тестовые сценарии на Gherkin (Vanessa-Automation)
- `scripts/` — Python-скрипты оркестрации
- `docker/` — Dockerfile и конфигурация контейнера с платформой 1С
- `reports/` — артефакты прогонов тестов (не коммитится)
- `docker-compose.yml` — композиция сервисов для локального запуска
- `.gitlab-ci.yml` — конфигурация CI/CD-конвейера
